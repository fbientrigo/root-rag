# Weekly Notes

Purpose:
- Track weekly evidence/report runs before claims are promoted.

Guidelines:
- Link each weekly note to `evidence/<run-id>/` and `reports/<run-id>_...`.
- Keep unresolved findings explicit and actionable.
- Do not mark workflow internals as `CONFIRMED` without source anchors.

Recommended note sections:
- Run metadata
- Candidate confirmed claims
- Unresolved questions
- Proposed query seeds
