# DISParticles

## Role
The `DISParticles` branch contains the outgoing particles produced in a Muon Deep Inelastic Scattering interaction. It is generated by `makeMuonDIS.py` and consumed by `MuDISGenerator`.

## Confirmed Facts
- **Tree Name**: `DIS`
- **Data Type**: `TClonesArray` of `TVectorD`.
- **Index Mapping**:
    - `[0]`: **PDG Code** (did)
    - `[1]`: **PX** (GeV/c)
    - `[2]`: **PY** (GeV/c)
    - `[3]`: **PZ** (GeV/c)
    - `[4]`: **Energy** (GeV)
- **Injection**: Injected into the simulation by `MuDISGenerator::ReadEvent`. The first particle in the array carries the DIS cross-section as its track weight.

## Source Anchors
- `muonDIS/makeMuonDIS.py:263-264`: TVectorD initialization.
- `shipgen/MuDISGenerator.cxx:191-209`: Loop and `AddTrack` usage.
